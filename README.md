# 📂 Projeto: Segmentação Comercial - Neoenergia
> **Status:** 🟢 Bloco 1 Concluído (Fundamentos de Python)

---

## 🎯 Objetivo do Projeto
Este projeto simula um cenário real na **Neoenergia Campinas**, onde utilizei Python para automatizar a análise de consumo de clientes da área comercial.

## 💼 O Desafio de Negócio
O objetivo é identificar perfis de consumo para direcionar campanhas de marketing:
* **Baixo Consumo:** Foco em eficiência energética (Kits LED).
* **Alto Consumo:** Foco em Upgrade de Plano e Energia Solar.

---

## 🛠️ Tecnologias e Hard Skills
* **Linguagem:** Python 3
* **Bibliotecas:** Pandas, Matplotlib e Seaborn
* **Lógica de Negócio:** Automação com `if/elif/else` e cálculo de KPIs.
